One backend,
four interfaces.

Whether you want a polished UI or a quick terminal command, MangaDL has you covered.

Web Dashboard

The flagship interface. Open http://localhost:5000 in any browser. Splash screen, animated cards, live progress, themes, history, cart.

$ MangaDL.exe

Desktop GUI

Native Tkinter desktop window — no browser required. Sidebar navigation, treeview chapters, live download polling.

$ python gui.py

Terminal UI

Beautiful colorful terminal interface using Rich. Gradient ASCII banner, live progress bars, color-coded sources.

$ python tui.py

Command Line

Scriptable, automation-ready. Perfect for cron jobs and shell scripts.

$ MangaDL-CLI download <url> --format pdf --mode volume

Frequently asked.

Everything you wanted to know about MangaDL.

Is it free?
Yes — completely free and open source under the MIT license. No premium tier, no microtransactions, no ads.
Do I need to install Python?
If you use the pre-built EXE/binary releases, no. They're fully self-contained. Only needed if you want to run from source or contribute.
Is it legal?
The tool itself is legal. How you use it is your responsibility. Respect copyright laws in your country and support creators by buying official manga when available.
Does it work offline?
The app runs locally and your downloaded files work offline forever. The download itself obviously requires an internet connection to fetch the manga.
Can I add my own source?
Yes — extend the BaseSource class in sources/base.py, implement three methods, and register it. See the README for a full guide.
Why so many themes?
Because not everyone reads in the dark. The high-contrast modes are WCAG AAA certified for users with visual impairments.
What's the difference between CBZ, PDF, and EPUB?
CBZ — best for comic readers like YACReader. PDF — universal, works anywhere. EPUB — best for e-readers like Apple Books, Kobo. Images — raw files for custom workflows.
Does it collect any data?
No. Zero telemetry. Zero analytics. Zero tracking. Your downloads, history, and settings stay on your machine.
Will it work with Kindle?
Sideload CBZ or EPUB files via Send to Kindle or USB. Native MOBI support is on the roadmap.
How do I update?
Download the new release and replace the old EXE. Your downloads folder, history, and settings persist between versions.
